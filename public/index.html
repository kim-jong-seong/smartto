스마또

고민하는 당신들을 위한 영리한 선택

스마또는 일상 속 크고 작은 결정들을 더 쉽고 재미있게 만들어주는 실시간 의사결정 플랫폼입니다. 혼자 고민하지 말고, 함께 만드는 현명한 선택을 경험해보세요.

왜 스마또인가요?

공정한 선택

무작위 알고리즘을 통한 편견 없는 의사결정

실시간 상호작용

여러 명이 동시에 참여하는 실시간 경험

스크래치 재미

복권 긁기 같은 재미있는 결과 확인 방식

간편한 사용

번거로운 회원가입 없이 즉시 시작

스마또 활용 가이드

1. 역할 배분

  • 게임에서 마피아/시민 역할 분배
  • 팀 프로젝트의 파트 배분
  • 조별 활동의 역할 분담
  • 발표 순서 정하기

2. 자리/위치 배정

  • 회의/식사 자리 배치
  • 발표/공연 순서 배정
  • 조별 좌석 배정
  • 담당 구역 분배

3. 무작위 선택

  • 상품/상자 배분
  • 경품 당첨자 선정
  • 무작위 팀 구성
  • 제비뽑기 대체

어떻게 사용하나요?

방 만들기

  1. 메인 화면에서 '방 만들기' 버튼을 클릭합니다.
  2. 자동으로 생성된 4자리 방 코드를 친구들과 공유합니다.
  3. 방장으로서 참여 인원과 선택 항목을 설정할 수 있습니다.
  4. 모든 참여자가 준비되면 게임을 시작합니다.

방 참여하기

  1. 친구에게 받은 4자리 방 코드를 입력합니다.
  2. 참여 후 방장의 게임 시작을 기다립니다.
  3. 게임이 시작되면 스크래치를 통해 결과를 확인합니다.
  4. 모두의 결과가 공개되면 게임이 종료됩니다.

스마또 사용 팁

  • 인원이 많을수록 더 재미있어요! 방장이 설정한 최대 인원까지 참여할 수 있습니다.
  • 결과는 스크래치를 통해 확인할 수 있어요. 긁는 재미를 느껴보세요!
  • 중복되지 않은 항목이 당신의 결과입니다.
  • 방장이 나가면 자동으로 다음 참여자에게 방장 권한이 넘어갑니다.
  • 모바일에서도 PC에서도 동일하게 즐길 수 있어요.
